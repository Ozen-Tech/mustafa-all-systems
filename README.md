# Aplicativo de Gestão de Promotores

Sistema completo para gestão de promotores com aplicativo mobile e dashboard web para supervisores.

## Estrutura do Projeto

- `mobile/` - App React Native para promotores
- `backend/` - API Node.js/Express
- `web/` - Dashboard React para supervisores
- `shared/` - Tipos TypeScript compartilhados

## Instalação

```bash
npm run install:all
```

## Desenvolvimento

```bash
# Backend
npm run dev:backend

# Mobile
npm run dev:mobile

# Web
npm run dev:web
```

## Tecnologias

- **Mobile:** React Native + Expo
- **Backend:** Node.js + Express + TypeScript + PostgreSQL + Prisma
- **Web:** React + TypeScript + Vite
- **Storage:** AWS S3
